# Virus Propagation Simulation Narrative

## Problem

Understanding spread dynamics requires balancing model realism with tractable computation and clear interpretation.

## My Role

I designed and implemented graph and probability-based simulation logic.

## Constraints

- Need interpretable outputs for scenario comparison
- Need adjustable parameters for intervention testing
- Need computationally manageable model complexity

## Key Decisions

1. Represent transmission context as graph structures
2. Use stochastic components to model uncertainty
3. Build result views that support strategy comparison

## Failure and Lesson

Some early outputs were technically correct but hard to interpret.
I improved visualization and scenario framing.
Lesson: modeling value is limited if results are not understandable by decision-makers.

## Outcomes

- Scenario-level visibility into spread and control strategies
- Stronger algorithmic reasoning through probabilistic simulation
- Practical experience translating model outputs into decision insights
